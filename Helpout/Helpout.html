Want to help out with this knowledgebase? Look no further! Getting started is super easy.

Fork the repository on Github

Head to github.com/LocalMonero/Knowledge and click "Fork" on the Github. If you don't have a Github account, it will prompt you to create one, which is pretty easy!

Editing the page directly in Github

Once that's complete, find a page you want to edit.html and open it. It should look something like this; then you can click Edit:

Copy everything in the file and heading to htmledit.squarefree.com, so you can somewhat see your changes in real time. Keep in mind, any images or pretty syntax will not show up. If you want them, you'll have to download the project locally and use Git like a normal person!

Once you're done, paste the changes back into Github and scroll to the bottom. Enter a commit message that describes what you changed, and tick "Create new branch" so you can seamlessly send a merge request with it:

Sending in your change

Now that your commit is completed, you can send in your change! Click "Compare across forks" to get started.

In the next screen, you'll want to select the left repository to be LocalMonero's. Click on it and select LocalMonero/Knowledge from the list:

Lastly, click "Create Pull Request" and a moderator will see it merged! You can do this as many times as you'd like to as many different files as you'd like. If you want to do more complex editing locally, such as images, or larger changes, feel free to do that as well (although it's out of the scope of this tutorial for now!).

Thank you for contributing to LocalMonero!
