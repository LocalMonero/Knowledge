Hosting anonymity enhancing nodes make the Monero network stronger. Here's a guide on what you can do!

I2P-Zero

I2P, or "Invisible Internet Project", is a decentralized darknet built in 2003 to protect users' privacy online. Today, it is leveraged by Monero to further strenghen anonymity in the payment network.

I2P-Zero is a user-friendly, lightweight router to the I2P privacy network. Running I2P-Zero helps strengthen the I2P network, and in turn helps Monero users send transactions over I2P securely as well. Running an I2P node is extremely lightweight both bandwidth and system resource-wise, and is a great way to contribute without even noticing.

Pruned & Full Nodes

Monero transactions are stored on an obfuscated ledger; running a Monero node means you keep track of this ledger too.

There's two types of nodes you can run; a full node, and a pruned node. Full nodes keep track of the full Monero history, whereas pruned nodes throw out a large portion of the data it doesn't need to keep. If you can run a full node it's recommended, but the blockchain size is currently 85GB and that's not feasible for everyone, so a pruned node is the next best thing you can do.
