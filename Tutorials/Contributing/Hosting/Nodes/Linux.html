Linux Full/Pruned Node Tutorial

  • Step 1: Download the latest Monero software for Linux: https://web.getmonero.org/downloads/
  • Step 2: Extract it into the target directory
  • Step 3: Run ./monerod --rpc-bind-port 18081 --restricted-rpc
    • Or, run a pruned (lightweight node): ./monerod --rpc-bind-port 18081 --restricted-rpc --prune-blockchain
You are now running a full node and maintaining a copy of the ledger.

Test your node:

  • Step 1: Go to canyouseeme.org/ and enter port 18080. If you can see yourself, great! You've configured your node correctly
If you can't see your node...

That means we need to port-forward on your router manually. This is unusual, but don't worry! It's not that hard either.

  • Step 4: Go to your Router page: http://192.168.0.1/
  • Step 5: Find your default router login/password from this website here: https://portforward.com/router-password/
  • Step 6: Using the default username found in the previous step, and your current public WiFi password, log into your router
  • Step 7: Run ifconfig to find your local IP address. Within the output, you are looking for something like this
    • If you do not have ifconfig installed, see this page for common distribution instructions.
  • Step 8: Port forward on port 18080. If you need help navigating your router page, see: https://portforward.com/router.htm
  • Step 9: Confirm your router is live! Head to https://canyouseeme.org/ and check port 18081.
